# mise plugin

Initializes [mise](https://github.com/jdx/mise), dev tools, env vars, task runner

To use it, add `mise` to the plugins array in your `.bashrc` file:

```bash
plugins=(... mise)
```

**Note:** you have to [install mise](https://mise.jdx.dev/getting-started.html#_1-install-mise-cli) first.
